NICE DCV SDK Demo (www.ni-sp.com)

NICE DCV is a high-end remote 2D/3D desktop product. Background on NICE DCV and NICE DCV Tips and Tricks.

Please enter the URL for the DCV server to connect to: (e.g. https://SERVER:8443) and hit return.

After entering the URL you will be prompted for username and password in the form below. (We will use the credentials only to login - please check the source code in case or feel free create your own copy).

After clicking "Login" the connection to your DCV server will be established - this will take a few seconds so please be patient. In the meantime you may monitor the javascript console (F12) for some messages.
